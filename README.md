# Detex Fraud Detector

A minimalist, high-contrast monochrome fraud detection tool.

## Features
- AI-powered fraud analysis (via OpenRouter/GPT)
- Real-time text classification (Safe, Suspicious, Fraud)
- Dual-panel sidebar navigation
- Persistent Dark/Light mode preferences
- Firebase Authentication & Firestore integration

## Local Setup
1. Clone the repository.
2. Create a `.env` file (or set your keys in `app.js`).
3. Open `index.html` in a local server.
